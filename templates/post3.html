PasswaordVault

This is an application that helps its users store all of their passwords in a single location on their computer. This program is based on the command line interface, and has a certain number of commands that the user can enter to manage their personal vault. This vault of passwords has a single master password that the user sets to access all of their passwords.

This program has the abilty to always be running in the background of your computer so that you can quickly look up your passwords for various online services.

Skills and Technologies Used:
  • - Python
  • - Hashing Framework
  • - SQLite3 Database Framework
  • - Data Structures
  • - Personal Password Suggestion Algorithm Design
