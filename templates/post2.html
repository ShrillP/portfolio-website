MindSpace

This is a program that helps its users reduce the mental stresses that they might have. It does this by incorporating various methods to help the user with their mental stress (e.g. meditation, a scheduler for their tasks, and a suggestion area where they can take a look at reputable sources and get information). It helps the user meditate using the pre-loaded files. The program also allow users to search for suggestions on various ways to decrease the effects of their mental stress. This program also has a task manager that will keep track of what they need to do.

Key Features
  • - Sign-Up section (multiple users)
  • - Specialized calendar for task management
  • - Allows for Google search queries right from UI
  • - Personalized messages for the user

Skills and Technologies Used:
  • - Java
  • - Control Structures
  • - Object Oriented Programming
  • - Web Technologies (JQuery)
  • - File IO
