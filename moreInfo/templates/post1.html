A* Pathfinding Algorithm Visualizer

This is a project that helps its users understand how the A* pathfinding algorithm works. The A* pathfinding algorithm is an algorithm that finds the shortest path between 2 nodes. The algorithm works to find a path that is the shortest and lowest cost (in terms of weights).

This tool allows the user to pick a start point, an end point, and lets them draw in their own barriers. Once the algorithm starts, the user can see how the algorithm approaches to find the shortest path to the end node.

With a clean UI design and simple controls, a user of any experience can learn from this tool.

Skills and Technologies Used:

  • - Pygame and TKinter Framework
  • - Object Oriented Programming
  • - Data Structures
  • - Algorithm Design
